leetcode-java
========

My Java solutions for "Interview Questions Online Judge" in the leetcode.com.

## Requirements
* Run: Java >= 1.6
* Compile: Java >= 1.6, Gradle >= 1.8.6

## Generate a file for a question

<pre>
$ gradle question "-Pq=Some Question"
</pre>

## Build

<pre>
$ gradle build
</pre>
